If you are going to do something on a regular basis, why not automate it?

This is a relational SQL database product from Microsoft. There are various different editions, which cater for the needs of different types of user - some editions being free. Generally, if an office uses Windows and office tools, Excel, Word etc, this would be the most obvious choice and one readily installable by a company's I.T. dept'.

A postgres database is an example of an open-source, cross-platform (runs on Linux, Windows and Apple) relational SQL database.

A mongo database is an open-source, cross-platform example of what is often called a NoSQL, or Document database. They are ideally suited to handling large volumes of rapidly changing, structured, semi-structured and unstructured data, which relational SQL databases aren't. They use JSON like documents with schemas.

In all cases data can be pulled from multiple sources and manipulated in seconds to produce the accurate information needed to write monthly accounting journals and feed into reports produced from, for example, Excel spreadsheets. Other reports can be written and run to check the information produced against controls, thus ensuring that everything is auditable.
